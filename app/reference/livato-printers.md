# Livato Printer Models - Product Nomenclature

## Desktop Printers (2-4 inch width)

### PrintPro Series (Desktop)
- **PrintPro 200** - 2" Compact Desktop Printer, 203 DPI
- **PrintPro 400** - 4" Desktop Printer, 203 DPI
- **PrintPro 400+** - 4" Desktop with LCD Display, 203 DPI
- **PrintPro 400HD** - 4" Desktop Printer, 300 DPI
- **PrintPro 400HD+** - 4" Desktop with Touch Display, 300 DPI

## Industrial Printers (4-6 inch width)

### IndustrialMax Series
- **IndustrialMax 4S** - 4" Standard Industrial, 203 DPI
- **IndustrialMax 4HD** - 4" High-Definition Industrial, 300 DPI
- **IndustrialMax 6S** - 6" Wide Format Industrial, 203 DPI
- **IndustrialMax 6HD** - 6" Wide Format HD, 300 DPI

## Mobile Printers (2-4 inch width)

- **MobilePrint 2BT** - 2" Bluetooth Mobile Printer
- **MobilePrint 3WiFi** - 3" WiFi Mobile Printer

## High-Speed Industrial

### VelocityPrint Series
- **VelocityPrint 8** - 8" High-Speed Industrial, 203 DPI
- **VelocityPrint 8HD** - 8" High-Speed Industrial, 300 DPI

## Nomenclature Guide

### Series Code
- **D** = Desktop
- **I** = Industrial
- **M** = Mobile
- **R** = RFID
- **H** = High-Speed

### Width Code (First Digit)
- **2** = 2 inch
- **3** = 3 inch
- **4** = 4 inch
- **6** = 6 inch
- **8** = 8 inch

### Resolution Code (Last Digit)
- **0** = 203 DPI
- **1** = 203 DPI (alternate)
- **2** = 300 DPI
- **3** = 300 DPI (alternate)

### Special Suffixes
- **T** = Touch Screen
- **W** = WiFi
- **B** = Bluetooth

## Usage in Datasheets

For shipping labels: **LV-D420, LV-D430, LV-I410**
For pharmaceutical: **LV-D430T, LV-I420**
For logistics: **LV-I610, LV-H810**
For retail: **LV-D200, LV-M200**
